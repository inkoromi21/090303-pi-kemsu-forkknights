# Fork Knights

Добавь своего героя через **fork** и **pull request**.

## 🚀 Как добавить героя

1. **Форкни** репозиторий → кнопка `Fork`.
2. **Склонируй** свой форк.
3. **Создай ветку** (можно работать в `main`, но лучше отдельную):
git checkout -b feature/твоя-фамилия
4. **Создай файл** в папке своей группы (`Group1`, `Group2` или `Group3`).  
Имя файла: `Фамилия.cs`.  
Класс должен:
- реализовывать `IKnight`;
- содержать **минимум 5 полей/свойств** (инициализировать в конструкторе);
- метод `GetJobApplication()` возвращает строку-объявление о работе (используй `$"..."` и эмодзи).
5. **Проверь** — запусти проект, твой герой должен появиться в консоли.
6. **Закоммить** и **запушь**.
7. **Создай pull request** из своей ветки в `main` этого репозитория.
8. **Создай Issue** с описанием героя (и, при желании, предложением по улучшению).

## ❓ Вопросы

Смотри пояснения в **методичке** или задавай в **Issues**.

Удачи!
